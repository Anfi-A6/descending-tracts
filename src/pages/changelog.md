# Changelog

## v1.0.0 Primera versión funcional del sitio

- Se muestran 8 niveles del tronco del encefálo:
  - Médula oblongada (Bulbo raquídeo)
    - Decusación de las pirámides
    - Decusación de los lemniscos
    - Núcleos olivares
    - Por debajo del puente
  - Puente
    - Colículo facial
    - Núcleos del trigémino
  - Mesencefálo
    - Colículo inferior
    - Colículo superior
- En cada nivel se puede interactuar con la imagen para mostrar el nombre de la estructura seleccionada
- En cada nivel se muestra una lista de todas las estructuras mencionadas por la bibliografía
- Barra de navegación
- Encabezado
